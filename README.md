# Odin Recipes 

Este projeto faz parte do currículo de fundamentos de desenvolvimento web do **The Odin Project**. 

##  Descrição
O site foi construído 100% com **HTML**, focado puramente na estrutura e sem o uso de estilos (CSS). O objetivo foi consolidar o aprendizado sobre a semântica da linguagem, organização de diretórios e navegação entre páginas.

##  Tecnologias Utilizadas
* **HTML5**: Estruturação de conteúdo, listas e links.
* **Git/GitHub**: Controle de versão e deploy via GitHub Pages.

##  O que foi desenvolvido
* Uma página inicial com links para as receitas.
* Páginas individuais para cada receita.
* Cada receita inclui descrição, ingredientes em listas não ordenadas e modo de preparo em listas ordenadas.
